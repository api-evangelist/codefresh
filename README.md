# CodefreshThis is a repo for managing machine readable artifacts for the [Codefresh APIs](https://codefresh.io). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Codefresh APIs](https://codefresh.io).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](https://codefresh.io)- [Developer](https://codefresh.io)- [Twitter](https://twitter.com/codefresh)- [GitHub](https://github.com/codefresh-io)The [APIs.json](https://github.com/api-evangelist/codefresh/blob/master/apis.json) index provides links to all the relevant links for the Codefresh APIs, including the OpenAPI and Postman if they are present.
